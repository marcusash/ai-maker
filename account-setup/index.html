GitHub Account Setup

Three paths. Most Microsoft employees only need Path 1.

Path 1 — Default (everyone)

Use your Microsoft enterprise account

Your alias_microsoft account is already provisioned. The AI Maker install guide works with this account out of the box. Nothing extra to do.

  1. Sign in to github.com with your Microsoft credentials. Your account looks like marcusash_microsoft.
  2. Run the AI Maker installer. Done — GitHub Copilot is included with your Microsoft license.
⋮
Path 2 — Optional (Microsoft employees)

Also link a personal GitHub account

Want to use AI Maker for personal projects, open source, or side work? You can link a personal GitHub account to your Microsoft identity — at no cost to you — and switch between the two freely.

  1. Create a personal GitHub account at github.com/signup if you don’t have one. Use your personal email, not your Microsoft email.
  2. Go to copilot.github.microsoft.com/link and sign in with your Microsoft corporate account.
  3. Authorize your personal GitHub account when prompted. This creates the link between your Entra ID and your personal GitHub.
  4. You’ll see a confirmation: “You are linked to [your personal account].” Done.
  5. In the GitHub Copilot app, sign out and sign back in with your personal account to switch between them.
What linking does: Your personal GitHub account joins the Microsoft Copilot org. This grants GitHub Copilot on your personal account, centrally funded by Microsoft. Your personal account stays yours — if you leave Microsoft, the license is removed but your account and repos are unaffected.
⋮
Path 3 — Not at Microsoft

Create a personal account

Using this public repo outside of Microsoft? You can still run AI Maker with a personal GitHub account.

  1. Create a free account at github.com/signup.
  2. Sign up for GitHub Copilot at github.com/features/copilot.
  3. Run the AI Maker installer and sign in with your personal account.
