GitHub Account Setup

Three paths. Most Microsoft employees only need Path 1.

PATH-1.EXE
Path 1 — Default (everyone) Start here
Use your Microsoft enterprise account

Your alias_microsoft account is already provisioned. The AI Maker installer works with this account out of the box. Nothing extra to do.

1
Sign in to GitHub
Go to github.com and sign in with your Microsoft credentials. Your account looks like <alias>_microsoft.
2
Run the AI Maker installer
Follow the install guide. When prompted to authorize GitHub Copilot, sign in with this account. Done.
· · ·
PATH-2.EXE
Path 2 — Optional (Microsoft employees)
Also link a personal GitHub account

Want to use AI Maker for personal projects, open source, or side work? You can link a personal GitHub account to your Microsoft identity. Both accounts have Copilot access — switch by signing out and back in to choose which account is active.

1
Create a personal GitHub account
Go to github.com/signup if you don’t have one. Use your personal email — not your Microsoft email.
2
Open the Microsoft link portal
Go to copilot.github.microsoft.com/link and sign in with your Microsoft corporate account.
3
Authorize your personal account
When prompted, authorize your personal GitHub account. This creates the link between your Entra ID and your personal GitHub.
4
Confirm the link
You’ll see a confirmation: “You are linked to [your personal account].” Done.
5
Switch between accounts
In the GitHub Copilot app, sign out and sign back in with your personal account to switch. Both accounts work with AI Maker.
What linking does: Your personal GitHub account joins the Microsoft Copilot org. Your personal account stays yours — your repos and history are unaffected by your employment status.
· · ·
PATH-3.EXE
Path 3 — Not at Microsoft
Create a personal account

Using this public repo outside of Microsoft? You can still run AI Maker with a personal GitHub account.

1
Create a GitHub account
Go to github.com/signup and create a GitHub account.
2
Set up GitHub Copilot
Visit github.com/features/copilot to choose a plan and set up Copilot on your account.
3
Run the AI Maker installer
Follow the install guide and sign in with your personal account when prompted.
