Install Guide

Get AI Maker running in about 10 minutes.

No command line experience needed. Seven steps from start to finish. You will copy and paste commands directly from this page.

🖥️ Windows 10 or 11
🌐 Internet connection
🐙 A GitHub.com account
⏱️ About 10 minutes
1

Extract the zip file

Right-click ai-maker-safe.zip and choose Extract All. Extract it to a location you will remember, like your Desktop or Documents folder.

After extracting, you will have an ai-maker-safe folder containing docs and scripts folders. Keep note of where this folder lives.

2

Run setup.bat

Inside the extracted folder, double-click setup.bat. It will ask for administrator permission, then automatically rename the scripts and launch the installer.

ℹ️

The scripts are packaged as .txt to pass Microsoft email and SharePoint filters. setup.bat handles the rename for you automatically before launching.

When Windows asks "Do you want to allow this app to make changes to your device?" click Yes. Administrator access is required to install software.

3

Open an Admin terminal in the scripts folder

In File Explorer, open the scripts\ai-maker folder. Click the address bar at the top of the window and copy the full path.

Next, right-click the Start button and choose Terminal (Admin). Click Yes when Windows asks for permission. Then type cd " then paste the path you copied, then type a closing " and press Enter. The quotes are required if your path has spaces.

Running as Admin lets the installer set everything up without interrupting you for permission prompts.

4

Run the installer

Copy and paste this command, then press Enter:

PowerShell
PowerShell -ExecutionPolicy Bypass -File .\install.ps1

The installer sets up everything: Node.js, Git, GitHub CLI, your AI Maker workspace, and a shortcut on your Desktop. This takes 3 to 5 minutes.

5

Sign in to GitHub when prompted

A browser window will open automatically. Enter your GitHub Enterprise username — your Microsoft alias followed by _microsoft — and click Sign in with your identity provider. Your Microsoft SSO handles the rest.

Do not close the browser tab. Once you approve, it will close on its own and the install will continue in PowerShell.

6

All 15 checks should say PASS

At the end of the install, you will see a list of 15 verification checks. Every one should show PASS. If they all pass, you are done.

✅

15 / 15 PASS: AI Maker is ready.

Look for this message at the bottom of your PowerShell window. When you see it, the install is complete and you can close PowerShell.

7

Launch AI Maker from your Desktop

Find the AI Maker icon on your Desktop — it looks like this: — and double-click it to launch.

Two things open at once: a chat window where you talk to AI Maker, and a browser tab called Canvas where your work comes to life. Type your questions in the chat window.

💡

The very first time it opens, it may ask: "GitHub Copilot CLI is not installed. Would you like to install it?" Press Y and Enter. This happens only once.

🎉

You're in. Welcome to AI Maker.

That was the hard part. From here, AI Maker handles the rest. A getting started guide will open in your browser — it walks you through your first conversation and introduces Canvas, one of AI Maker's most useful features.

Something not working?

PowerShell says "cannot be loaded because running scripts is disabled"

Run this command first, then try Step 4 again: Set-ExecutionPolicy -Scope CurrentUser RemoteSigned and press Y when asked.

The installer fails with a winget or network error

If Node.js, Git, and the GitHub CLI are already installed on your machine, try running .\install.ps1 -SkipPrereqs instead. This skips the software install steps.

I closed PowerShell before seeing 15/15 PASS

Open a new PowerShell window, navigate back to the scripts folder (Step 4), and run .\test.ps1 to see which steps passed.

The scripts still have a .txt extension and won't run

Run setup.bat again from the extracted folder. It is safe to run more than once.
