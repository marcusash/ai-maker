> WHAT IS AI MAKER?
AI Maker is a personalized GitHub Copilot agent that learns how you work and gets better at helping you over time. It runs in your terminal via the Copilot CLI.
One setup script. One agent. Done in five minutes. No GitHub repo required, no dev environment to configure.
If you have a Microsoft 365 Copilot license, AI Maker also connects to your email, calendar, and Teams via WorkIQ.
> PREREQUISITES
Required
  • Windows 10 or Windows 11
  • Node.js 22+ and PowerShell 7+ (the setup script installs these if missing)
Microsoft Employees
  • Personal GitHub account: If you don't have one, create at github.com/signup
  • Enterprise account: Use format yourAlias_microsoft (e.g., marcusash_microsoft). GitHub Copilot is included.
External Users
  • GitHub account: github.com/signup
  • GitHub Copilot Pro or higher ($10/month). The free plan does not support the CLI. Subscribe here.
WorkIQ (Optional : Microsoft 365 users only)
  • An organizational Microsoft 365 account (E3, E5, Business Standard, or Business Premium)
  • Microsoft 365 Copilot add-on license
  • IT admin consent to WorkIQ in your organization's Entra ID
  • Personal Outlook.com accounts do not qualify
> QUICK START
Open PowerShell and run:
irm https://raw.githubusercontent.com/marcusash/ai-maker/main/setup.ps1 | iex
The script will ask for your name, then copy the AI Maker agent files to your machine and personalize them. If you want WorkIQ, it will walk you through that too.
> DETAILED INSTALLATION STEPS
1
Get a GitHub account
If you don't have one: github.com/signup
2
Subscribe to GitHub Copilot Pro
$10/month minimum. The free plan does not support custom agents or CLI access.
Subscribe: github.com/features/copilot/plans
3
Open PowerShell
Press Win+X and select "Terminal" or "PowerShell".
Or: search "PowerShell" in Start menu and click it.
4
Run the setup script
irm https://raw.githubusercontent.com/marcusash/ai-maker/main/setup.ps1 | iex
The script will:
  • Ask for your name
  • Install Node.js and PowerShell 7 if needed
  • Copy AI Maker agent files to your machine
  • Personalize the agent with your name
  • Optionally set up WorkIQ for M365 integration
5
Sign in to GitHub
The setup script will prompt you to authenticate with GitHub. Follow the instructions in your terminal. This is a one-time step.
6
Launch AI Maker
After setup completes, open the Copilot CLI from the Start menu or desktop shortcut. AI Maker is ready.
7
Run the onboarding interview
In the Copilot CLI, say: Good morning.
AI Maker will run the onboarding interview. Answer one question at a time. This teaches AI Maker your working style.
8
Start working
AI Maker is now available everywhere. No switching. No context loss. Just open the CLI and start chatting.
> WHAT YOU GET
AI Maker agent : a personalized Copilot agent with skills for design, code, research, quality, data, brainstorming, and user research
Onboarding interview : AI Maker learns your working style in the first session
WorkIQ integration : if you have M365 Copilot, AI Maker can query your calendar, email, and Teams without you having to summarize them
Personalization : agent knows your name and adapts to your preferences over time
> TROUBLESHOOTING
Script execution error
If you see "running scripts is disabled", run this first:
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
GitHub CLI not found
The setup script should install GitHub CLI automatically. If you see this error, manually install from:
cli.github.com
Node.js version error
AI Maker requires Node.js 22 or higher. The setup script will install it if missing. If you have an older version, uninstall it first from Settings > Apps, then re-run the setup script.
