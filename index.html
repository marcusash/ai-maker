AI-MAKER.EXE
> WHAT IS AI MAKER?
Your AI assistant for research, writing, brainstorming, data analysis, design, and communication — in a real window, not a terminal. One agent, 11 skills, five minutes to set up.
Runs inside the GitHub Copilot App. Works everywhere. No admin rights required.
> SETUP GUIDE
1
What you need
  • Windows 11
  • Internet connection
  • GitHub account — sign in with your alias_microsoft enterprise account (e.g. marcusash_microsoft). The install guide works with this account out of the box.
Microsoft employee? You can also link a personal GitHub account for personal projects — free within Microsoft. Learn how →  •  Not at Microsoft? Create a personal account at github.com
Upgrading from the old CLI setup? Migration guide →
2
Install
Open PowerShell (Win+X → Terminal) and paste this command:
>> PASTE INTO POWERSHELL
curl.exe -sSL -o $env:TEMP\install.bat https://github.com/marcusash/ai-maker/releases/download/v3.0.5/install.bat; & $env:TEMP\install.bat
ⓘ  Installs the GitHub Copilot App, 11 AI skills, and Agency. Takes ~5 minutes. No admin rights required.
ⓘ  Nothing is pushed to GitHub — your workspace stays local in C:\GitHub\ai-workspace.
3
Open the GitHub Copilot App
The app opens automatically after install. Or search “GitHub Copilot” in the Start menu and sign in with your GitHub account when prompted.
4
Add your project folder
In the App's left rail, find the Sessions section and click the + button next to it. Choose Add local folder and select C:\GitHub\ai-workspace. This is where your AI agent lives — the folder was created during install.
5
Start your first session
Open a new session in the App and say hello. AI maker greets you and asks what you’re working on. All 11 skills are always active — just describe what you need.

> WHAT YOU GET
One AI agent with 11 dedicated skills. Ask it anything — it picks the right skill automatically.
Canvas
Ask for "a canvas" and AI maker builds a live visual in your browser — dashboards, status reports, slide content. No PowerPoint required.
Vault
"Save this to my vault" — AI maker writes it to a file on your PC, organized and findable. Your context survives across sessions.
WorkIQ / read M365
Read your Microsoft 365. Ask "what's on my calendar?" or "what tasks are overdue?" AI maker reads and summarizes — no changes made.
Skills
11 built-in skills: research, design, writing, code review, data analysis, brainstorming, user research, canvas, vault, ops, and WorkIQ. Just describe what you need.
Agency / act on M365
Act on your Microsoft 365. Send emails, create meetings, post to Teams, add Planner tasks — all with your approval. Entra auth built in, no extra config.
> SOMETHING WRONG?
The installer didn’t finish or something looks off
It’s safe to run again — the installer detects what’s already in place and skips those steps.
irm https://github.com/marcusash/ai-maker/releases/download/v3.0.5/reset.bat -OutFile $env:TEMP\reset.bat; & $env:TEMP\reset.bat
