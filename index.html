> INITIALIZING AI MAKER
One agent. Works everywhere. Five minutes to set up.
You stay in your world. Everything you know is still true. You just get an AI partner that makes your work better.
> WHAT YOU GET
One AI agent for everything you do. Research, design, writing, data analysis, brainstorming, code review, and communication in one partner. No switching. No context loss.
Canvas
Ask AI Maker to "open a canvas" and it builds a visual in your browser. Project dashboards, status reports, slide decks. No PowerPoint required.
Vault
Your personal notes folder. Say "save this to my vault" and AI Maker writes it to a file on your PC, organized and always easy to find.
WorkIQ
Connected to Microsoft 365. Ask "what is on my task list?" or "pull my calendar for next week." Works alongside your existing M365 workflow.
Skills
8 built-in skills: research, design, writing, code review, data analysis, brainstorming, user research, and communication. Just describe what you need.
> PREREQUISITES
Microsoft Employees
1. Personal GitHub account: If you don't have one, create at github.com/signup
2. Enterprise account: Use format yourAlias_microsoft (e.g., marcusash_microsoft). GitHub Copilot is included.
External Users
GitHub Copilot Pro required ($10/month). Subscribe here.
WorkIQ (Optional: Microsoft 365 users only)
An organizational M365 account with Copilot add-on. IT admin consent to WorkIQ in Entra ID required. Personal Outlook.com accounts do not qualify.
> INSTALLATION
1
Open Windows Terminal as Admin
Press Win+X, select "Terminal (Admin)"
Or search "Terminal" in Start menu, right-click, "Run as administrator"
2
Paste this command and press Enter
irm https://raw.githubusercontent.com/marcusash/ai-maker/main/bootstrap.ps1 | iex
The script installs Git if needed, downloads AI Maker, and sets everything up. Troubleshooting: if you see "running scripts is disabled", run Set-ExecutionPolicy -Scope CurrentUser RemoteSigned first.
3
Follow the prompts
The script asks for your name, signs you in to GitHub, and optionally sets up WorkIQ for M365. Answer each prompt and press Enter.

> YOU'RE SET UP. HERE'S WHAT TO DO.
AI Maker learns through the work. After 3-5 sessions, it adapts to your tone, your decision-making style, your priorities. Start here.
Step 1: Launch your agent
Double-click AI Agents on your desktop (uses this GitHub icon). The GitHub CLI opens with AI Maker ready.
Step 2: Start your first session
Copy this prompt, switch to the terminal window, paste it, and press Enter. AI Maker will introduce itself and ask you a few questions to get started.
I'm new here. Please introduce yourself, then ask me a few questions to learn my name, my role, my team, and what I'm working on right now.
After the onboarding, AI Maker greets you by name from session two onward. It knows your preferences, your biggest time drains, and how you want feedback delivered.
