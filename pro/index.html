> WHAT IS AI MAKER?
For power users who want version control and the full technical skill set. Two AI agents, 22 skills, private GitHub repo that backs up your work, and restores it on any machine in minutes.
Everything the Blue Pill gives you, plus AI workbench, GitHub backup, and cross-machine restore.
> SETUP GUIDE
1
What you need
  • Windows 11
  • Internet connection
  • GitHub account, sign in with your alias_microsoft enterprise account (e.g. marcusash_microsoft). The install guide works with this account out of the box.
Microsoft employee? You can also link a personal GitHub account for personal projects, free within Microsoft. Learn how →  •  Not at Microsoft? Create a personal account at github.com
Git, GitHub CLI, and VS Code are installed automatically.
2
Install
Open PowerShell (Win+X → Terminal) and paste this command. When prompted, choose option 2 (Red Pill):
>> PASTE INTO POWERSHELL
curl.exe -sSL -o $env:TEMP\install.bat https://github.com/marcusash/ai-maker/releases/download/v3.0.5/install.bat; & $env:TEMP\install.bat
3
Authenticate with GitHub
The installer opens a browser window and prompts you to sign in to GitHub. This creates your private ai-workspace repo and pushes your first backup. One-time step.
4
Open the GitHub Copilot App
The app opens automatically after install. Or search “GitHub Copilot” in the Start menu.
When asked to sign in:
  • Go to github.com and sign in with your enterprise credentials, your alias_microsoft account
  • The app will show you a code, enter it on GitHub when prompted to authorize the device
  • Follow the prompts to complete the connection
5
Add your project folder
In the App's left rail, find the Sessions section and click the + button next to it. Choose Add local folder and select C:\GitHub\ai-workspace. This is where both AI agents live, the folder was created during install.
6
Start your first session
Open a new session in the App and say hello. AI maker greets you and asks what you’re working on. If it’s technical, AI maker spins up AI workbench in a new session automatically. All 22 skills are always active.
7
Try these. Click to copy, then paste into AI maker.
Or press Win+H to activate voice and speak the prompt, add any context you need out loud.

Play devil’s advocate on a decision I’m about to make. (then speak your plan)
BRAINSTORM · VOICE
I have a hard 1:1 this week. Help me prepare, what to say, what to ask, what to avoid.

> RESTORE ON A NEW MACHINE
Your ai-workspace folder is a private GitHub repo. To restore on a new PC:
1. Run the installer on your new machine (same command). Choose option 2.
2. Authenticate with GitHub. The installer detects your existing ai-workspace repo and restores everything automatically.
3. Open the App, add C:\GitHub\ai-workspace as a project. Your agents remember everything.
> SOMETHING WRONG?
The installer didn’t finish or something looks off
It’s safe to run again, the installer detects what’s already in place and skips those steps.
irm https://github.com/marcusash/ai-maker/releases/download/v3.0.5/reset.bat -OutFile $env:TEMP\reset.bat; & $env:TEMP\reset.bat
