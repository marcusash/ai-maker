> WHAT IS AI MAKER PRO?
For power users who want version control and the full technical skill set. Two AI agents, 22 skills, private GitHub repo that backs up your work — and restores it on any machine in minutes.
Everything the Blue Pill gives you, plus AI Workbench, GitHub backup, and cross-machine restore.
> SETUP GUIDE
1
What you need
  • Windows 10 or 11
  • Internet connection
  • GitHub account (free) — create one at github.com/signup
Microsoft employees: use your alias_microsoft enterprise account. Git, GitHub CLI, and VS Code are installed automatically by the Red Pill installer.
2
Install
Open PowerShell (Win+X → Terminal) and paste this command. When prompted, choose option 2 (Red Pill):
>> PASTE INTO POWERSHELL
curl.exe -sSL -o $env:TEMP\install.bat https://github.com/marcusash/ai-maker/releases/download/v3.0.0/install.bat; & $env:TEMP\install.bat
ⓘ  Installs the GitHub Copilot App, 22 AI skills (AI Maker + AI Workbench), Agency, Git, and GitHub CLI. Takes ~10 minutes.
ⓘ  Creates a private {username}/ai-workspace repo on GitHub. Your vault and settings back up there automatically.
3
Authenticate with GitHub
The installer opens a browser window and prompts you to sign in to GitHub. This creates your private ai-workspace repo and pushes your first backup. One-time step.
4
Open the GitHub Copilot App
The app opens automatically after install. Or search “GitHub Copilot” in Start. Sign in with your GitHub account when prompted.
5
Add your project folder
In the App's left rail, find the Sessions section and click the + button next to it. Choose Add local folder and select C:\GitHub\ai-workspace. This is where both AI agents live — the folder was created during install.
6
Start your first session
Open a new session in the App and paste this prompt:
I'm new here. Please introduce yourself, then ask me a few questions to learn my name, my role, my team, and what I'm working on right now.
After onboarding, your agents greet you by name each session. All 22 skills are always active — just describe what you need.

> RESTORE ON A NEW MACHINE
Your ai-workspace folder is a private GitHub repo. To restore on a new PC:
1. Run the installer on your new machine (same command). Choose option 2.
2. Authenticate with GitHub. The installer detects your existing ai-workspace repo and restores everything automatically.
3. Open the App, add C:\GitHub\ai-workspace as a project. Your agents remember everything.
> SOMETHING WRONG?
The installer didn’t finish or something looks off
It’s safe to run again — the installer detects what’s already in place and skips those steps.
irm https://github.com/marcusash/ai-maker/releases/download/v3.0.0/reset.bat -OutFile $env:TEMP\reset.bat; & $env:TEMP\reset.bat
